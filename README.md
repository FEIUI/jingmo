## 介绍

『京墨』开源、免费、公益性 APP。献给喜欢中华文字、文学、文化的人。

目前的内容有：

1. 古诗词文 10000 首（阙、篇）；

2. 古诗词文名句 10000 句；

3. 歇后语 14026 条；

4. 成语 30895 条；

5. 中国传统色 161 种；

6. 中国传统节日 19 个；

7. 二十四节气；

8. 绕口令 45 段；

9. 知识卡片 464 组；

数据来自网络，或有内容错误，欢迎纠错，请在公众号留言。
