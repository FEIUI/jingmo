## 介绍

『京墨』开源的古诗词文（名句）、歇后语、成语、传统节日、绕口令等的阅读APP

献给喜欢中华文字、文学、文化的人。

目前的内容有：

1. 诗文 110 万篇,包含诗、词、辞赋、残句、联、文等；

2. 诗文名句 10000 句；

3. 歇后语 14026 条；

4. 成语 30895 条；

5. 中国传统色 161 种；

6. 中国传统节日 19 个；

7. 二十四节气；

8. 绕口令 45 段；

9. 知识卡片 464 组；

数据来自网络，或有内容错误，欢迎纠错，请在公众号留言。
